# Harbor Vault Specification Index

Harbor Vault is a Business Operating System built on the Harbor Model.

The product is AI native, organization centered, and designed around one question:

What should I do next?

## Foundation Rule

Do not build screens before the module specification answers:

- What is this screen for?
- What data belongs here?
- What action should the user take next?
- What should Harbor AI do here?
- Which roles can see or change this?
